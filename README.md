# gaffer

```
Usage: gaffer [options] context ontology gaf outfile

  --catalog <value>  Catalog file for ontology loading (optional)
  context            Path to JSON-LD context file
  ontology           Ontology IRI
  gaf                Path to GAF
  outfile            Path to output JSON
```
